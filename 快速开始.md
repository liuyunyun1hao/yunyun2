# 🚀 快速开始 - 5分钟部署指南

## 准备工作
1. GitHub账号
2. 硅基流动API密钥（从硅基流动官网获取）

## 一键部署

### 方法A：使用Render蓝图（最简单）
1. **上传到GitHub**
   - 将本项目所有文件上传到GitHub仓库
   
2. **一键部署**
   - 访问 [render.com](https://render.com)
   - 登录 → "New +" → "Blueprint"
   - 选择你的仓库 → "Apply"

### 方法B：手动部署
1. **创建Render服务**
   ```
   名称: yunyun-proxy
   环境: Python
   构建命令: pip install -r requirements.txt
   启动命令: python proxy_server_render.py
   ```

2. **设置环境变量**
   ```
   REQUIRE_AUTH: true
   AUTH_PASSWORD: [设置强密码]
   ```

## 获取服务地址
部署完成后，Render会提供：
```
https://yunyun-proxy.onrender.com
```

## 首次配置
1. **获取密码**
   - 在Render控制台 → Environment → 查看AUTH_PASSWORD

2. **导入API密钥**
```bash
curl -X POST 'https://yunyun-proxy.onrender.com/api/data' \
  -H 'Authorization: Basic $(echo -n admin:你的密码 | base64)' \
  -H 'Content-Type: application/json' \
  -d '{"keys":[{"key":"sk-你的密钥","balance":"未知","fail_count":0}],"active_key":"sk-你的密钥"}'
```

## 立即使用
```bash
curl -X POST 'https://yunyun-proxy.onrender.com/v1/chat/completions' \
  -H 'Authorization: Basic $(echo -n admin:密码 | base64)' \
  -H 'Content-Type: application/json' \
  -d '{
    "model": "Qwen/Qwen2.5-7B-Instruct",
    "messages": [{"role": "user", "content": "你好"}]
  }'
```

## 💡 提示
- 用户名固定为 `admin`
- 密码在Render环境变量中查看
- 首次请求可能较慢（冷启动约30秒）
- 15分钟无活动会自动休眠
